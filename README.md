# AI Automation Portfolio

## Project 1: AI Chat Assistant

### Overview

This workflow uses n8n and an AI model to answer user questions automatically through a chat interface.

### Workflow

User Message → AI Model → Response

### Tools Used

- n8n
- OpenRouter
- AI Language Model

### Business Use Case

A small business could use this workflow as the foundation for:

- Customer support chatbot
- FAQ assistant
- Website AI assistant
- Internal employee helpdesk

### Skills Demonstrated

- Workflow automation
- AI integration
- API credential setup
- Prompt engineering
- n8n workflow design
