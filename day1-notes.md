# Day 1 Notes

An AI-powered chat workflow in n8n that receives a user message and generates a response using an AI model through OpenRouter.

## What I Learned

- Creating workflows in n8n
- Connecting AI models

- Testing workflow execution

## Real Business Use Case

A digital marketing agency could use this workflow to answer common client questions automatically before a human agent gets involved.
